에너지 균형 방정식

에너지 균형 방정식은 다음과 같습니다.

에너지 균형 = 에너지 입력 - 에너지 출력

방정식만 보면 복잡하지는 않지만 에너지가 들어오는 양과 에너지가 나가는 양을 계산하는 과정에서 귀찮음이 있죠.

하지만 에너지 균형 방정식을 구하여 벌크업이나 다이어트를 할때, 많은 도움이 됩니다.

체중 증가 = 에너지 입력 > 에너지 출력

체중 감소 = 에너지 입력 < 에너지 출력

이 식에서 에너지 입력 양을 계산하는 방법은 자신이 섭취한 음식의 칼로리들을 더하면 됩니다.

하지만 칼로리 출력 양을 계산하기 위해선 자신의 하루 칼로리를 계산하고 하루 칼로리를 구하기위한 활동 대사량 또한 구해야합니다.

이제 하나씩 알아보도록 합시다.

Basal Metabolic Rate (BMR)

"BMR" 이란 에너지를 섭취한 것을 숨쉬기, 혈액순환 등 몸의 기본적인 기능을 수행하기 위해 필요한 에너지로 전환시키는 비율을 말합니다.

즉, 기초적인 생명활동이나 신진대사를 위해 사용이 되고 저절로 소모되는 에너지 이기 때문에 기초 대사량을 높일 수록 비교적 살이 덜 찐다고 알려져 있습니다.

기초대사율을 구하는 방법에는 "Harris benedict", "Mifflin St.Jeor", "Katch-McArdle" 공식 등등이 있다고 합니다.

계속 찾아봤는데, "Harris benedict"나 "Mifflin St.Jeor"에 다루는 것만 많고 다른 공식에 대해선 다루지는 않는것 같았습니다.

- Harris benedict equation

이 방정식은 1919년에 나왔으며, 체중과 키, 나이를 대입해 BMR을 구하고, 활동 계수를 적용하여 총 일일 에너지 소비 칼로리를 계산하게 됩니다.

중학교나 고등학교 책에 나온 기억에는 없지만 많이 다뤄지는 것 같았어요.

(Men) BMR = 66.473 + ( 13.7516 × weight in kg ) + ( 5.0033 × height in cm ) – ( 6.755 × age in years )

(Women) BMR = 655.0955 + ( 9.5634 × weight in kg ) + ( 1.8496 × height in cm ) – ( 4.6756 × age in years )

아래는 "Harris benedict"의 신뢰성 검증 연구들 입니다.

    Frankenfield DC, et al. Prediction of resting metabolic rate in critically ill patients. Crit Care Med. 2003 Aug;31(8):2012-7.

    Hall KD, et al. Quantification of the effect of energy imbalance on bodyweight. Lancet. 2011 Aug 27;378(9793):826-37.

이 공식은 1990년도에 "Mifflin St.Jeor"이 개정한 아래 방정식으로 나오게 됩니다.

- Mifflin St.Jeor equation

이 식 또한 기존 "Harris benedict"방정식을 개선한것과 같이 키, 체중, 나이를 토대로 BMR을 계산하고, 신체활동 지수를 적용해 일일 칼로리 소모량을 추정할 수 있습니다.

(Men)BMR = ( 10 × weight in kg ) + ( 6.25 × height in cm ) – ( 5 × age in years ) + 5

(Women)BMR = ( 10 × weight in kg ) + ( 6.25 × height in cm ) – ( 5 × age in years ) – 161

이 방정식은 고등학교 체육 책에서 자주 보였던 기초대사율 계산법 이네요.

- Katch-McArdle

"Katch McArdle"은 자주 보이지 않고 Mifflin St.jeor과 비교했을 대 TDEE 추정치가 다르며, 보통 Mifflin St.jeor을 사용한다고 하기 때문에 이런게 있다, 라고만 보시면 될것 같습니다.

이 공식은 체지방량(Lean Body mass)을 고려하는 공식으로, 체지방량을 알고 있다면 더욱 정확한 BMR을 계산할 수 있다고 합니다.

BMR = 370 + ( 21.6 * Lean Body Mass in kg )

(Men)LBM = ( 0.407 x weight in kg ) + ( 0.267 x height in cm ) - 19.2

(Women)LBM = ( 0.252 x weight in kg ) + ( 0.473 x height in cm ) - 48.3

위와 같은 방정식들은 BMR을 구하는 방정식의 일부에 불과하고 실제 적용할 때는 개인의 특성과 상황을 고려해서 사용하는것이 중요합니다.
